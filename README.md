# Anki Hero

An Electron-based application for managing and reviewing Anki flashcards with enhanced features and improved user experience.

## Features

- Deck management and organization
- Enhanced review interface
- Import/export functionality
- Advanced scheduling algorithms
- Modern, responsive UI

## Development

This project uses Electron with TypeScript and React for the frontend.

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

## Project Structure

- `electron/` - Main process code
- `preload/` - Preload scripts
- `src/` - Renderer process (React app)
- `dist-*` - Build outputs
- `types/` - TypeScript type definitions
